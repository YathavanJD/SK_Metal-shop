SKMetal Shop – E-Commerce Web Application

SKMetal Shop is a web-based e-commerce application developed as a group project to manage and sell metal products online. 
The system allows users to browse products, view product details, and place orders through
a simple and user-friendly interface.

 Features

Browse and view metal products
Product detail pages
Admin product management system
User-friendly shopping interface
Responsive website design

Technologies Used

HTML
CSS
JavaScript
PHP
MySQL

 My Contribution

Developed the Product Page functionality for both Admin and Customer interfaces
Implemented features to view product details and manage products
Assisted in integrating product data with the database

How to Run

Clone the repository
Import the database into MySQL
Run the project using XAMPP / WAMP server
Open the project in your browser

Purpose
This project was developed as a group assignment to gain experience in full-stack web development, 
database integration, and collaborative software development.
